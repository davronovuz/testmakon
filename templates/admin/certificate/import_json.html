Import sozlamalari

Tanlasangiz, JSON dagi subject_slug e'tiborga olinmaydi.

Belgilansa, import qilinadigan mock bepul bo'ladi.

Belgilansa, bir xil slug bilan mavjud mock va savollar yangilanadi.

Bekor qilish

JSON format

Fayl quyidagi tuzilishga ega bo'lishi kerak:

{
  "subject_slug": "matematika",
  "mock_slug": "mat-2024-1",
  "mock_title": "Matematika 2024 — 1-variant",
  "year": 2024,
  "version": "1-variant",
  "time_limit": 150,
  "questions": [
    {
      "number": 1,
      "type": "choice",
      "text": "2 + 2 = ?",
      "points": 1,
      "difficulty": "easy",
      "topic_slug": "arifmetika",
      "explanation": "To'g'ri javob B) 4",
      "choices": [
        {"label":"A","text":"3","is_correct":false,"order":1},
        {"label":"B","text":"4","is_correct":true,"order":2},
        {"label":"C","text":"5","is_correct":false,"order":3},
        {"label":"D","text":"6","is_correct":false,"order":4}
      ]
    },
    {
      "number": 33,
      "type": "grouped_af",
      "text": "Kvadrat funksiyalarni moslashtiring.",
      "points": 3,
      "difficulty": "medium",
      "topic_slug": "kvadrat-funksiyalar",
      "options": [
        {"label":"A","text":"$(1,0)$","order":1},
        {"label":"B","text":"$(2,-1)$","order":2},
        {"label":"C","text":"$(0,3)$","order":3},
        {"label":"D","text":"$(-1,2)$","order":4},
        {"label":"E","text":"$(-2,5)$","order":5},
        {"label":"F","text":"$(3,1)$","order":6}
      ],
      "items": [
        {"number":33,"text":"$y=(x-1)^2$","correct_option":"A"},
        {"number":34,"text":"$y=(x-2)^2-1$","correct_option":"B"},
        {"number":35,"text":"$y=x^2+3$","correct_option":"C"}
      ]
    },
    {
      "number": 34,
      "type": "short_open",
      "text": "x = ?",
      "points": 2,
      "difficulty": "hard",
      "topic_slug": "tenlamalar",
      "explanation": "x = 3 chunki ...",
      "answer": {"value":"3","type":"integer","tolerance":0}
    },
    {
      "number": 35,
      "type": "multi_part",
      "text": "a=5, b=3",
      "points": 4,
      "difficulty": "medium",
      "topic_slug": "arifmetika",
      "parts": [
        {"label":"a","text":"a+b=?","points":1,
         "answer":"8","answer_type":"integer","tolerance":0}
      ]
    }
  ]
}

Savol turlari:

type Tavsif
choice A/B/C/D tanlov. choices massiv kerak.
grouped_af A–F variantlar. options + items kerak.
short_open Ochiq javob. answer ob'ekti kerak.
multi_part Ko'p qismli. parts massiv kerak.

Ixtiyoriy maydonlar (har bir savol uchun):

Maydon Qiymatlar
difficulty "easy" | "medium" | "hard" (default: medium)
topic_slug Mavzu slug (tests_app.Topic.slug). Topilmasa e'tiborga olinmaydi.
explanation Javob tushuntirishi (string)
hint Maslahat matni (string)
